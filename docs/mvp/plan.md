## MVP Scope and Plan (Centralized)

### Scope
- Teacher onboarding, course editor (video + quiz + tool plugin), course market
- Parent discovery, purchase, family learning space, progress tracking
- Developer tool listing, basic install into courses, usage metering
- Payments（PSP）、revenue split（teacher/dev/platform）、basic refunds
- Ratings/reviews、off-chain achievements（badge）
- Moderation pipeline（text/image）、parental consent and safety baseline

### Milestones
- M0 (8–10 weeks): Prototype web + API + payments sandbox + 1–2 tools
- M1 (10–14 weeks): Private beta (10 teachers, 100 families, 3 tools), payouts
- M2 (+12 weeks): Public beta, expanded recommendation and moderation features

### Metrics
- Supply: live courses, teacher retention, tool integrations
- Demand: new families, first-purchase conversion, weekly learning retention
- Commerce: GMV, revenue split distribution, refund rate, repurchase rate
- Safety/Quality: moderation pass rate, violation rate, appeal SLA, NPS
