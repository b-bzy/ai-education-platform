## Website IA & Routes

### Global
- / — Landing (value prop, roles, featured courses/tools)
- /auth — Login/Signup (Parent, Teacher, Developer)
- /search — Search (courses, teachers, tools, posts)
- /help — Help Center, Safety, Compliance

### Parents
- /parents/home — Personalized feed
- /parents/courses — Discovery with filters (age, topic, difficulty, rating)
- /parents/library — Purchased/Enrolled
- /parents/child/[id] — Child profile, progress, milestones
- /parents/community — Posts, Q&A, topics
- /parents/orders — Orders, refunds

### Teachers
- /teacher/dashboard — KPIs, revenue, ratings
- /teacher/courses — List, create, edit
- /teacher/courses/new — Editor (video, quizzes, tool plugins)
- /teacher/marketing — Coupons, trials, affiliates
- /teacher/partners — Tool co-creation & integrations
- /teacher/payouts — Withdrawals, split rules

### Developers
- /dev/dashboard — Usage, installs, feedback
- /dev/tools — List, create, edit
- /dev/tools/new — Tool listing + SDK doc
- /dev/sandbox — Test harness
- /dev/billing — Pricing, metering, payouts

### Public Profiles
- /t/[handle] — Teacher profile & catalog
- /tool/[slug] — Tool listing & reviews
- /course/[slug] — Course detail, syllabus, preview

### Admin/Moderation
- /admin/moderation — Queue, actions, appeals
- /admin/governance — Proposals, votes
